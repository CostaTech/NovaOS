# NovaOS

NovaOS is a new clean operating system project by CostaTech.

Goal: a real bootable live OS using Assembly, C and C++.

## Language roles

- Assembly: boot entry and CPU setup
- C: low-level drivers and hardware access
- C++: desktop, apps, filesystem and higher-level systems
- NovaC: future app/compiler layer

## Build

```bash
make iso
```

## Run in QEMU

```bash
make run
```
