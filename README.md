# NovaOS

NovaOS is a new clean operating system project by CostaTech.

Goal: a real bootable live OS using Assembly, C and C++.

## Language roles

- Assembly: boot entry and CPU setup
- C: low-level drivers and hardware access
- C++: desktop, apps, filesystem and higher-level systems
- NovaC: future app/compiler layer
# NovaOS

## About

NovaOS is a new operating system built completely from scratch.

Unlike many other operating systems, NovaOS does not use or reuse source code from existing operating systems. Every component is designed and developed specifically for this project, making it a unique and independent system.

NovaOS features its own programming language, created to give developers a native way to interact with the system and build applications specifically for its environment.

Inspired by projects like TempleOS, NovaOS aims to explore the idea of creating a complete ecosystem where the operating system, programming language, and user experience are designed together as one unified vision.

## Features

- 🖥️ Custom operating system built from zero
- ⚙️ Independent kernel architecture
- 💻 No reused source code from other operating systems
- 🔤 Custom programming language
- 🚀 Unique ecosystem designed from the ground up
- 📚 Created for learning, experimentation, and innovation

## Vision

NovaOS is not just an operating system; it is an experiment in creativity and computer science.

The goal is to create something completely original, where every part of the system has its own purpose and identity.
